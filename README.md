# Backrooms

Backrooms game written in Three.js, with procedural maze generation, monster AI, and mobile support.

**Live demo:** https://szabolevi98.github.io/backrooms

## Controls

| Input | Action |
|---|---|
| `WASD` | Move |
| Mouse | Look |
| `Left Shift` | Sprint |
| `C` | Crouch (duck under hanging walls) |
| `M` | Mute / Unmute |
| `ESC` | Pause |
